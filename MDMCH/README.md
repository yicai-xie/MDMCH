# DCMH

Deep cross-modal hashing 论文 pytorch 实现

## 环境

`python 3.5+`

`pytorch 0.3.0+`

## 运行

`mkdir checkpoints result` 创建相关文件夹

`python main.py help` 查看帮助信息

`python main.py train` 训练并测试

`python main.py test` 单独进行测试
